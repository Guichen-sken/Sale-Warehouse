# 📦 Sale Warehouse

> **A curated collection of digital assets, production-ready projects, and creative experiments.**

本项目是一个结构化的仓库，用于管理我个人开发的不同阶段的项目资产。通过 Git 分支进行分类管理，确保每一个项目都能在合适的场景下展示其价值。

---

## 📂 仓库结构说明

本仓库采用**分支驱动 (Branch-Driven)** 的管理模式，请根据需求切换至相应分支查看代码：

### 1. [💰 Sold (已售)](https://github.com/Guichen-sken/Sale-Warehouse/tree/sold)
*   **用途**: 存放所有已经成功交付、成交的项目存档。
*   **状态**: 归档/只读。
*   **价值**: 作为过往成功案例的证明，展示工程质量和业务落地能力。

### 2. [🏷️ For Sale (在售)](https://github.com/Guichen-sken/Sale-Warehouse/tree/for-sale)
*   **用途**: 存放当前正在对外展示、寻求合作或直接购买的成熟项目。
*   **状态**: 持续维护/演示中。
*   **价值**: 包含完整的产品说明、技术栈介绍及部署指南。

### 3. [🎨 Creative (创意)](https://github.com/Guichen-sken/Sale-Warehouse/tree/creative)
*   **用途**: 存放个人灵感、实验性技术 Demo 以及尚未完全成型的潜力项目。
*   **状态**: 活跃开发/实验中。
*   **价值**: 体现对新技术的好奇心与探索精神，是创意的发源地。

---

## 🚀 如何使用

1.  **克隆仓库**:
    ```bash
    git clone https://github.com/Guichen-sken/Sale-Warehouse.git
    ```
2.  **切换分支**:
    ```bash
    # 查看在售项目
    git checkout for-sale

    # 查看创意灵感
    git checkout creative
    ```
